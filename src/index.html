¿Qué es el DOM?

DOM = Document Object Model

Es una representación en memoria de tu página HTML. Convierte cada elemento HTML en un objeto que JavaScript puede manipular.

Imagina que tu página HTML es como un árbol familiar:

  • El document es la raíz (abuelo)
  • Los elementos como <html>, <body> son las ramas (padres)
  • Los elementos dentro de ellos como <p>, <div> son las hojas (hijos)

Del HTML al DOM

Código HTML:

<!DOCTYPE html>
<html>
<head>
    <title>Mi Página</title>
</head>
<body>
    <h1 id="titulo">Hola</h1>
    <p>Este es un párrafo</p>
</body>
</html>

Estructura del DOM:

¿Por qué es importante? El DOM permite a JavaScript:

  • Cambiar el texto y contenido de la página
  • Modificar estilos y clases CSS
  • Reaccionar a eventos (clics, teclas presionadas, etc.)
  • Crear nuevos elementos o eliminar existentes

Selección de Elementos

Antes de modificar cualquier elemento, primero debemos seleccionarlo. Esto es como decirle a JavaScript: "Quiero trabajar con este elemento específico".

Selección por ID

El método getElementById() selecciona UN elemento que tenga el ID especificado.

Recuerda que los IDs deben ser únicos en tu página (no puede haber dos elementos con el mismo ID).

HTML:

<div id="miCaja">
    Soy una caja con ID "miCaja"
</div>

JavaScript:

// Seleccionamos el elemento con ID "miCaja"
const miElemento = document.getElementById('miCaja');
Esta caja tiene ID "cajaDemostracion"

Selección por Selector CSS (querySelector)

El método querySelector() es muy poderoso porque usa la misma sintaxis que CSS para seleccionar elementos.

Este método selecciona el primer elemento que coincida con el selector.

Recordatorio de selectores CSS:

  • #id - Selecciona por ID (ej. #miCaja)
  • .clase - Selecciona por clase (ej. .miClase)
  • etiqueta - Selecciona por tipo de etiqueta (ej. p, div)

HTML:

<p class="destacado">Párrafo 1</p>
<p class="destacado">Párrafo 2</p>
<p>Párrafo normal</p>

JavaScript:

// Selecciona el primer párrafo con clase "destacado"
const parrafo = document.querySelector('.destacado');

Este es el primer párrafo destacado

Este es el segundo párrafo destacado

Este es un párrafo normal

Selección múltiple (querySelectorAll)

El método querySelectorAll() selecciona todos los elementos que coincidan con el selector.

Devuelve una lista de elementos que podemos recorrer uno por uno.

HTML:

<p class="destacado">Párrafo 1</p>
<p class="destacado">Párrafo 2</p>
<p>Párrafo normal</p>

JavaScript:

// Selecciona TODOS los párrafos con clase "destacado"
const parrafos = document.querySelectorAll('.destacado');

// Podemos recorrerlos con un bucle
parrafos.forEach(parrafo => {
    console.log(parrafo.textContent);
});

Importante: querySelectorAll() devuelve algo similar a un array, pero no es exactamente un array. Es una "NodeList" que tiene el método forEach pero puede no tener otros métodos de arrays.

Manipulación de Contenido

Una vez seleccionado un elemento, podemos cambiar su contenido de diferentes maneras.

Cambiar texto con textContent

La propiedad textContent permite cambiar (o leer) el contenido de texto de un elemento.

Es como borrar todo lo que hay dentro del elemento y escribir nuevo texto plano.

HTML inicial:

<p id="mensaje">Texto original</p>

JavaScript:

// Seleccionamos el elemento
const mensaje = document.getElementById('mensaje');

// Cambiamos su contenido de texto
mensaje.textContent = '¡Nuevo texto!';

Este es el texto original

¿Cuándo usar textContent? Cuando quieras establecer solo texto sin ningún formato HTML. Es más seguro y rápido que innerHTML.

Cambiar HTML con innerHTML

La propiedad innerHTML permite cambiar el contenido HTML completo de un elemento.

Con innerHTML puedes incluir etiquetas HTML que serán interpretadas por el navegador.

HTML inicial:

<div id="contenedor">Texto normal</div>

JavaScript:

// Seleccionamos el elemento
const contenedor = document.getElementById('contenedor');

// Cambiamos su contenido HTML
contenedor.innerHTML = '<strong>Texto en negrita</strong> y <em>cursiva</em>';
Este es contenido normal

Precaución: innerHTML puede ser inseguro si introduces contenido proporcionado por usuarios sin filtrar, ya que podría permitir ataques de inyección de código.

Modificar atributos

Podemos cambiar cualquier atributo de un elemento HTML usando:

  • getAttribute() - Para obtener el valor de un atributo
  • setAttribute() - Para establecer o cambiar un atributo

HTML:

<img id="miImagen" 
     src="imagen1.jpg" 
     alt="Descripción original">

JavaScript:

// Seleccionamos la imagen
const imagen = document.getElementById('miImagen');

// Leemos un atributo
const srcActual = imagen.getAttribute('src');
console.log(srcActual); // "imagen1.jpg"

// Cambiamos un atributo
imagen.setAttribute('src', 'imagen2.jpg');
imagen.setAttribute('alt', 'Nueva descripción');

Manipulación de Estilos

JavaScript puede modificar el aspecto visual de los elementos cambiando sus estilos CSS directamente o manipulando sus clases.

Cambiar estilos directamente

La propiedad style permite modificar los estilos CSS de un elemento directamente desde JavaScript.

Es como añadir estilos en línea (inline styles) al elemento.

HTML:

<div id="miCaja">
    Una caja para cambiar estilos
</div>

JavaScript:

// Seleccionamos el elemento
const caja = document.getElementById('miCaja');

// Cambiamos varios estilos
caja.style.backgroundColor = 'lightblue';
caja.style.color = 'darkblue';
caja.style.padding = '20px';
caja.style.borderRadius = '10px';

Importante: Las propiedades CSS que tienen guiones como background-color se escriben en camelCase en JavaScript: backgroundColor.

Esta caja cambiará de estilo

Trabajar con clases CSS

La propiedad classList proporciona métodos para manipular las clases CSS de un elemento:

  • classList.add('nombre-clase') - Añade una clase
  • classList.remove('nombre-clase') - Elimina una clase
  • classList.toggle('nombre-clase') - Alterna una clase (la añade si no existe, la quita si existe)
  • classList.contains('nombre-clase') - Comprueba si tiene una clase

HTML y CSS:

/* CSS */
.destacado {
    background-color: yellow;
    font-weight: bold;
    padding: 10px;
}

/* HTML */
<p id="parrafo">Este párrafo cambiará</p>

JavaScript:

// Seleccionamos el elemento
const parrafo = document.getElementById('parrafo');

// Añadimos una clase
parrafo.classList.add('destacado');

// Comprobamos si tiene una clase
if (parrafo.classList.contains('destacado')) {
    console.log('Tiene la clase destacado');
}

// Alternamos una clase (la quita si existe)
parrafo.classList.toggle('destacado');

Este párrafo cambiará sus clases CSS

Ventaja: Manipular clases es mejor que cambiar estilos directamente porque:

  • Mantiene la separación entre CSS (estilos) y JavaScript (comportamiento)
  • Es más fácil de mantener y actualizar
  • Permite aplicar múltiples estilos a la vez

Aplicación práctica: Tema claro/oscuro

Podemos combinar lo aprendido para crear funcionalidades comunes como un selector de tema.

Selector de tema

Este es un ejemplo de cómo cambiar el tema de un componente usando classList.

// HTML
<div id="contenedor">
    <h4>Selector de tema</h4>
    <p>Este es un contenido de ejemplo.</p>
    <button id="botonTema">Cambiar a tema oscuro</button>
</div>

// CSS
.tema-oscuro {
    background-color: #333;
    color: #fff;
}

// JavaScript
const botonTema = document.getElementById('botonTema');
const contenedor = document.getElementById('contenedor');

botonTema.addEventListener('click', function() {
    // Alterna la clase tema-oscuro
    contenedor.classList.toggle('tema-oscuro');
    
    // Actualiza el texto del botón
    if (contenedor.classList.contains('tema-oscuro')) {
        botonTema.textContent = 'Cambiar a tema claro';
    } else {
        botonTema.textContent = 'Cambiar a tema oscuro';
    }
});

Eventos: Haciendo Páginas Interactivas

Los eventos son acciones que ocurren en la página web, como clics, movimientos del mouse, pulsaciones de teclas, etc. JavaScript puede "escuchar" estos eventos y ejecutar código cuando ocurran.

Eventos de clic básicos

El método addEventListener() conecta un evento con una función que se ejecutará cuando ocurra ese evento.

Sintaxis: elemento.addEventListener('tipoDeEvento', funcionAEjecutar);

HTML:

<button id="miBoton">Haz clic</button>

JavaScript:

// Seleccionamos el botón
const boton = document.getElementById('miBoton');

// Añadimos un "escuchador" de eventos
boton.addEventListener('click', function() {
    alert('¡Hiciste clic en el botón!');
});

También puedes asignar funciones ya definidas:

function mostrarMensaje() {
    console.log('Botón presionado');
}

boton.addEventListener('click', mostrarMensaje);

Eventos con información del evento

Cuando se ejecuta un manejador de eventos, recibe automáticamente un objeto event con información sobre el evento.

Este objeto contiene propiedades útiles como:

  • event.target - El elemento que desencadenó el evento
  • event.type - El tipo de evento (ej. "click")
  • event.preventDefault() - Método para prevenir el comportamiento predeterminado

HTML:

<a href="https://ejemplo.com" id="miEnlace">
    Enlace de ejemplo
</a>

JavaScript:

const enlace = document.getElementById('miEnlace');

enlace.addEventListener('click', function(event) {
    // Prevenir que el enlace abra la URL
    event.preventDefault();
    
    // Mostrar un mensaje en su lugar
    console.log('Clic en:', event.target.textContent);
    console.log('Href:', event.target.href);
});
Este enlace no te llevará a Mozilla

Eventos del ratón

Podemos detectar diferentes interacciones del ratón:

  • click - Clic simple
  • dblclick - Doble clic
  • mouseover - Cuando el ratón entra en el elemento
  • mouseout - Cuando el ratón sale del elemento
  • mousemove - Cuando el ratón se mueve dentro del elemento
Interactúa conmigo
// HTML
<div id="area">Interactúa conmigo</div>

// JavaScript
const area = document.getElementById('area');

area.addEventListener('mouseover', function() {
    this.style.backgroundColor = '#3498db';
    this.style.color = 'white';
    console.log('Ratón dentro');
});

area.addEventListener('mouseout', function() {
    this.style.backgroundColor = '';
    this.style.color = '';
    console.log('Ratón fuera');
});

// Más eventos
area.addEventListener('click', () => console.log('Clic'));
area.addEventListener('dblclick', () => console.log('Doble clic'));

Eventos de teclado

JavaScript puede detectar cuando el usuario presiona teclas:

  • keydown - Cuando una tecla se presiona
  • keyup - Cuando una tecla se suelta
  • keypress - Cuando se presiona una tecla que produce un carácter (deprecated)
// HTML
<input type="text" id="miInput" placeholder="Escribe aquí...">

// JavaScript
const input = document.getElementById('miInput');

// Detectar teclas presionadas
input.addEventListener('keydown', function(event) {
    console.log('Tecla presionada:', event.key);
    console.log('Código de la tecla:', event.code);
    
    // Detectar teclas específicas
    if (event.key === 'Enter') {
        console.log('¡Presionaste Enter!');
    }
});

Eventos de formulario

Los formularios tienen eventos especiales:

  • submit - Cuando se envía un formulario
  • reset - Cuando se restablece un formulario
  • change - Cuando cambia el valor de un input (al perder el foco)
  • input - Cuando cambia el valor (en tiempo real)
// HTML
<form id="miFormulario">
    <input type="text" id="nombre">
    <button type="submit">Enviar</button>
</form>

// JavaScript
const formulario = document.getElementById('miFormulario');
const inputNombre = document.getElementById('nombre');

// Evento al enviar el formulario
formulario.addEventListener('submit', function(event) {
    // Prevenir el envío normal del formulario
    event.preventDefault();
    
    // Hacer algo con los datos
    console.log('Formulario enviado con:', inputNombre.value);
});

// Evento en tiempo real mientras se escribe
inputNombre.addEventListener('input', function() {
    console.log('Valor actual:', this.value);
});

Creación y Eliminación de Elementos

JavaScript no solo puede modificar elementos existentes, también puede crear elementos completamente nuevos o eliminar elementos del DOM.

Crear elementos nuevos

Para crear elementos seguimos tres pasos:

  1. Crear el elemento con createElement()
  2. Configurar propiedades y contenido
  3. Añadir el elemento al DOM con appendChild() u otros métodos

HTML inicial:

<div id="contenedor">
    <!-- Aquí se añadirán elementos -->
</div>

JavaScript:

// Seleccionar contenedor
const contenedor = document.getElementById('contenedor');

// 1. Crear nuevo elemento
const nuevoParrafo = document.createElement('p');

// 2. Configurar contenido y propiedades
nuevoParrafo.textContent = 'Este párrafo fue creado con JavaScript';
nuevoParrafo.style.color = 'blue';
nuevoParrafo.classList.add('destacado');

// 3. Añadir al DOM
contenedor.appendChild(nuevoParrafo);

Creación de elementos complejos

Podemos crear estructuras más complejas anidando elementos.

// Creación de una tarjeta completa con JavaScript
function crearTarjeta(titulo, contenido) {
    // Crear elementos principales
    const tarjeta = document.createElement('div');
    const encabezado = document.createElement('h3');
    const texto = document.createElement('p');
    const boton = document.createElement('button');
    
    // Configurar contenido
    encabezado.textContent = titulo;
    texto.textContent = contenido;
    boton.textContent = 'Aceptar';
    
    // Configurar estilos
    tarjeta.classList.add('tarjeta');
    // Añadir más estilos...
    
    // Añadir evento al botón
    boton.addEventListener('click', function() {
        console.log('Tarjeta aceptada');
    });
    
    // Ensamblar la estructura
    tarjeta.appendChild(encabezado);
    tarjeta.appendChild(texto);
    tarjeta.appendChild(boton);
    
    // Devolver la tarjeta completa
    return tarjeta;
}

// Uso
const miTarjeta = crearTarjeta('Título de ejemplo', 'Contenido de la tarjeta');
document.getElementById('contenedor').appendChild(miTarjeta);

Eliminar elementos

Hay varias formas de eliminar elementos del DOM:

  • elemento.remove() - Elimina el elemento directamente
  • padre.removeChild(hijo) - Elimina un hijo desde el padre

Elemento 1

Elemento 2

Elemento 3

// Método 1: Eliminar un elemento directamente
const elemento = document.getElementById('miElemento');
elemento.remove();

// Método 2: Eliminar un hijo desde el padre
const padre = document.getElementById('contenedor');
const hijo = document.getElementById('elementoAEliminar');
padre.removeChild(hijo);

// Método 3: Eliminar todos los hijos
function eliminarTodosLosHijos(elemento) {
    while (elemento.firstChild) {
        elemento.removeChild(elemento.firstChild);
    }
}

eliminarTodosLosHijos(document.getElementById('contenedor'));

Insertar elementos en posiciones específicas

Además de appendChild, existen otros métodos para insertar elementos en diferentes posiciones:

  • padre.appendChild(elemento) - Añade al final
  • padre.insertBefore(nuevo, referencia) - Inserta antes de otro elemento
  • elemento.insertAdjacentElement(posición, elemento) - Inserta en una posición relativa
